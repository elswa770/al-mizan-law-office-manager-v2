# ⚖️ Al-Mizan Law Office Manager

نظام إدارة متكامل للمكاتب القانونية والقضايا، مبني بتقنيات حديثة لتسهيل إدارة العمل القانوني.

## ✨ المميزات

### 📋 **إدارة القضايا**
- إضافة وتعديل وحذف القضايا
- تصنيف القضايا حسب الحالة (نشطة، مغلقة، مؤرشفة)
- نظام أرشفة تلقائي للقضايا القديمة
- استعادة القضايا المؤرشفة

### 👥 **إدارة العملاء**
- سجل شامل للعملاء
- ربط العملاء بالقضايا
- معلومات التواصل والبيانات الشخصية

### 📅 **إدارة الجلسات**
- جدولة الجلسات والمواعيد
- تذكيرات بالجلسات القادمة
- تسجيل نتائج الجلسات

### 💰 **إدارة الماليات**
- تتبع الرسوم والمصروفات
- سجل المدفوعات المستلمة
- تقارير مالية شاملة

### 📄 **إدارة المستندات**
- رفع وتنظيم المستندات القانونية
- تصنيف المستندات حسب النوع
- معاينة وتحميل المستندات

### 🔐 **الأمان والصلاحيات**
- نظام تسجيل دخول آمن
- صلاحيات متعددة للمستخدمين
- حماية البيانات الحساسة

### ☁️ **التكامل مع السحابة**
- مزامنة تلقائية مع Firebase
- نسخ احتياطي تلقائي
- وصول من أي جهاز

## 🛠️ التقنيات المستخدمة

### **الواجهة الأمامية**
- **React 18** - واجهة مستخدم تفاعلية
- **TypeScript** - برمجة آمنة ومكتوبة
- **Tailwind CSS** - تصميم عصري ومتجاوب
- **Lucide React** - أيقونات حديثة

### **الخلفية والتخزين**
- **Firebase Firestore** - قاعدة بيانات NoSQL
- **Firebase Storage** - تخزين الملفات والصور
- **Firebase Authentication** - نظام المصادقة

### **أدوات التطوير**
- **Vite** - أداة بناء سريعة
- **ESLint** - جودة الكود
- **Git** - التحكم في الإصدارات

## 🚀 التثبيت والتشغيل

### **المتطلبات**
- Node.js 16 أو أحدث
- npm أو yarn
- حساب Firebase

### **خطوات التثبيت**

1. **نسخ المستودع**
   ```bash
   git clone https://github.com/elswa770/al-mizan-law-office-manager.git
   cd al-mizan-law-office-manager
   ```

2. **تثبيت الاعتماديات**
   ```bash
   npm install
   ```

3. **إعداد Firebase**
   - أنشئ مشروع جديد في [Firebase Console](https://console.firebase.google.com)
   - أنشئ تطبيق ويب
   - انسخ إعدادات Firebase إلى `services/firebaseConfig.ts`
   - قم بنشر قواعد Firestore من `firestore.rules`

4. **تشغيل المشروع**
   ```bash
   npm run dev
   ```

5. **فتح التطبيق**
   - اذهب إلى `http://localhost:5173`
   - قم بإنشاء حساب جديد أو تسجيل الدخول
