## 🚀 التثبيت والتشغيل

### **المتطلبات**
- Node.js 16 أو أحدث
- npm أو yarn
- حساب Firebase

### **خطوات التثبيت**

1. **نسخ المستودع**
   ```bash
   git clone https://github.com/elswa770/al-mizan-law-office-manager.git
   cd al-mizan-law-office-manager
   ```

2. **تثبيت الاعتماديات**
   ```bash
   npm install
   ```

3. **إعداد Firebase**
   - أنشئ مشروع جديد في [Firebase Console](https://console.firebase.google.com)
   - أنشئ تطبيق ويب
   - انسخ إعدادات Firebase إلى `services/firebaseConfig.ts`
   - قم بنشر قواعد Firestore من `firestore.rules`

4. **تشغيل المشروع**
   ```bash
   npm run dev
   ```

5. **فتح التطبيق**
   - اذهب إلى `http://localhost:5173`
   - قم بإنشاء حساب جديد أو تسجيل الدخول
