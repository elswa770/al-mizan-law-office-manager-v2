### 3. **دعم Offline الكامل**

#### **🔄 معالجة العمليات:**
```typescript
const handleSaveAppointment = async (e: React.FormEvent) => {
  try {
    if (isOnline) {
      // Online: Update directly
      if (editingAppointment) {
        onUpdateAppointment(appointmentData);
      } else {
        onAddAppointment(appointmentData);
      }
    } else {
      // Offline: Add to pending actions
      await offlineManager.addPendingAction({
        type: editingAppointment ? 'update' : 'create',
        entity: 'appointment',
        data: appointmentData
      });
      
      // Update local state immediately for better UX
      if (editingAppointment) {
        onUpdateAppointment(appointmentData);
      } else {
        onAddAppointment(appointmentData);
      }
    }
  } catch (error) {
    console.error('Error saving appointment:', error);
  }
};
```

#### **📊 مؤشرات الحالة:**
- **شريط إنذار** عند عدم الاتصال
- **عرض عدد المواعيد المعلقة** للمزامنة
- **أيقونة WiFi** في الهيدر تتغير حسب الحالة
